## Image Optimization Tips

1. **Compress images** before uploading (use TinyPNG, ImageOptim, etc.)
2. **Use appropriate formats**:
   - JPG for photos
   - PNG for logos with transparency
   - WebP for modern browsers (better compression)
3. **Recommended sizes** are maximum dimensions - images will scale down responsively
4. **Alt text** is already included in the Blade templates for accessibility
